  <name>Task 1: Implement extract() pipeline with upsert, scene linking, and run recording</name>
  <files>backend/app/services/breakdown_service.py</files>
  <action>
Replace the `extract()` stub in BreakdownService with the full pipeline. Add the following private methods and update `extract()`:

1. **`async def _call_ai_extraction(self, ctx: ExtractionContext) -> ExtractionResponse`**:
   - Import `chat_completion_structured` from `.ai_provider`
   - Build messages: system message with EXTRACTION_SYSTEM_PROMPT, user message from `self._build_user_prompt(ctx)`
   - Call `chat_completion_structured(messages=messages, response_model=ExtractionResponse, temperature=0.15, max_tokens=8000)`
   - Return the validated ExtractionResponse

2. **`def _upsert_elements(self, db: Session, project_id: UUID, extracted: List[ExtractedElement]) -> dict`**:
   - Pre-load ALL existing BreakdownElements for this project in a single query
   - Build lookup map: `{(category, name.lower()): element for element in existing}`
   - For each extracted element:
     a. Look up by `(item.category, item.canonical_name.lower())`
     b. If existing AND `is_deleted=True`: SKIP (SYNC-02 -- do not resurrect). Increment `skipped`.
     c. If existing AND `user_modified=True`: SKIP update but ADD to element_map for scene linking (SYNC-01). Increment `skipped`.
     d. If existing AND neither deleted nor user-modified: UPDATE description, set source="ai". Increment `updated`.
     e. If not existing: CREATE new BreakdownElement with project_id, category, canonical_name as name, description, source="ai". Call `db.add()` then `db.flush()` to get the ID. Increment `created`.
   - Track all non-skipped elements in `element_map: dict[str, BreakdownElement]` keyed by canonical_name
   - Return `{"created": N, "updated": N, "skipped": N, "element_map": element_map}`
   - CRITICAL: All UUID comparisons use `str()` cast

3. **`def _reconcile_scene_links(self, db: Session, element: database.BreakdownElement, new_scene_ids: List[str]) -> None`**:
   - Delete ALL existing AI-sourced scene links for this element: `db.query(ElementSceneLink).filter(element_id == str(element.id), source == "ai").delete(synchronize_session="fetch")`
   - For each scene_id in new_scene_ids:
     a. Check if a user-sourced link already exists for this (element_id, scene_item_id) pair -- if so, skip
     b. Otherwise create a new ElementSceneLink with element_id, scene_item_id=scene_id, context="", source="ai"
     c. `db.add(link)`

4. **`def _map_scene_indices_to_ids(self, ctx: ExtractionContext, scene_appearances: List[ExtractedSceneAppearance]) -> List[str]`**:
   - For each appearance, convert 1-based `scene_index` to 0-based index into `ctx.scene_summaries`
   - Return list of scene summary `id` values (as strings) for valid indices
   - Skip invalid indices (out of range) with a warning log

5. **`def _record_run(self, db: Session, project_id: UUID, status: str, result: dict, error: str = None, created: int = 0, updated: int = 0) -> database.BreakdownRun`**:
   - Create BreakdownRun with project_id, status, config={"temperature": 0.15, "provider": settings.AI_PROVIDER}, result_summary=result, error_message=error, elements_created=created, elements_updated=updated
   - Set completed_at=datetime.now(timezone.utc) if status in ("completed", "failed")
   - `db.add(run)` and return it

6. **Update `async def extract(self, db: Session, project_id: UUID) -> database.BreakdownRun`**:
   - Full pipeline in a single transaction:
     a. Build context: `ctx = self._build_extraction_context(db, project_id)`
     b. Validate context has screenplay content (if no screenplay_texts, record a "failed" run and return)
     c. Call AI: `response = await self._call_ai_extraction(ctx)`
     d. Upsert elements: `result = self._upsert_elements(db, project_id, response.elements)`
     e. Reconcile scene links for each element in result["element_map"]:
        - Find the matching ExtractedElement by canonical_name
        - Map scene indices to IDs via _map_scene_indices_to_ids
        - Call _reconcile_scene_links
     f. Record run: status="completed", created/updated counts from result
     g. `db.commit()` -- SINGLE commit for the entire transaction
     h. Return the BreakdownRun
   - Wrap in try/except: on any exception, `db.rollback()`, record a "failed" run, commit that, re-raise or return the failed run

Import `datetime` from datetime, `timezone` from datetime, and `settings` from `..config`.
