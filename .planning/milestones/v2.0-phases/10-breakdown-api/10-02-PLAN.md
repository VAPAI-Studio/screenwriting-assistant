**Three new endpoint groups:**

1. **POST /extract/{project_id}** (API-01 -- extraction trigger stub):
   - `response_model=schemas.BreakdownRunResponse`
   - Call `_verify_project_ownership`
   - Create a `database.BreakdownRun` with `project_id=project_id`, `status="pending"`, `config={}`, `result_summary={"message": "Extraction not yet implemented"}`
   - Commit, refresh, return the run
   - Docstring: "Trigger AI extraction. Extraction logic stubbed until Phase 11."
   - Return 200 (not 202 -- the stub is synchronous per research recommendation)

2. **Scene link add/remove** (API-06):

   a. `POST /element/{element_id}/scenes` (status_code=201):
   - Body: `schemas.SceneLinkCreate`
   - Call `_verify_element_ownership`
   - Validate scene exists: query `database.ListItem` by `body.scene_item_id`, raise 404 if not found
   - Check for existing link (idempotent): query `ElementSceneLink` by `element_id + scene_item_id`. If exists, return 200 with `{"message": "Scene link already exists", "id": str(existing.id)}`
   - Create `database.ElementSceneLink` with `element_id`, `scene_item_id=body.scene_item_id`, `context=body.context`, `source="user"`
   - Commit, refresh, return `{"message": "Scene linked", "id": str(link.id)}`

   b. `DELETE /element/{element_id}/scenes/{scene_item_id}`:
   - Path params: `element_id: UUID`, `scene_item_id: UUID`
   - Call `_verify_element_ownership`
   - Query `ElementSceneLink` by `element_id + scene_item_id`. If not found, raise 404
   - `db.delete(link)` (hard delete for junction table -- this is NOT a soft-delete entity)
   - Commit
   - Return `{"status": "success", "message": "Scene link removed"}`

3. **GET /summary/{project_id}** (API-07):
   - `response_model=schemas.BreakdownSummaryResponse`
   - Call `_verify_project_ownership`
   - Single aggregation query: `db.query(database.BreakdownElement.category, func.count(database.BreakdownElement.id)).filter(project_id, is_deleted==False).group_by(category).all()`
   - Convert to dict: `{cat: count for cat, count in counts}`
   - Sum for total_elements
   - Latest run: `db.query(database.BreakdownRun).filter(project_id).order_by(BreakdownRun.created_at.desc()).first()`
   - Return `schemas.BreakdownSummaryResponse(project_id=project_id, is_stale=project.breakdown_stale or False, total_elements=total_elements, counts_by_category=counts_dict, last_run=schemas.BreakdownRunResponse.model_validate(last_run) if last_run else None)`
