Implementation steps:
1. Call `_verify_element_ownership(db, element_id, current_user.id)` to get `element`.
2. Look up `story.characters` PhaseData: query `database.PhaseData` filtering `project_id=str(element.project_id)`, `phase="story"`, `subsection_key="characters"`.
3. If not found, create it: `database.PhaseData(project_id=element.project_id, phase="story", subsection_key="characters", content={}, sort_order=0)`, `db.add()`, `db.flush()` (NOT commit).
4. Load all existing ListItems for `chars_pd.id` via `db.query(database.ListItem).filter(database.ListItem.phase_data_id == str(chars_pd.id)).all()`.
5. Iterate existing items: if `(item.content.get("name") or "").lower() == element.name.lower()`, return `JSONResponse(status_code=status.HTTP_200_OK, content={"status": "already_exists", "list_item_id": str(item.id)})`.
6. Create `database.ListItem(phase_data_id=chars_pd.id, item_type="supporting", content={"name": element.name, "role": element.description or "", "dialogue_style": ""}, status="draft", sort_order=len(existing_items))`.
7. `db.add(new_item)`, `db.commit()`, `db.refresh(new_item)`.
8. Return `JSONResponse(status_code=status.HTTP_200_OK, content={"status": "created", "list_item_id": str(new_item.id)})`.
