**2. Create `frontend/src/components/Breakdown/AudioPlayer.tsx`:**

Imports: `useRef`, `useState`, `useEffect` from `react`, `Play`, `Pause`, `Square` from `lucide-react`.

```typescript
interface AudioPlayerProps {
  src: string;
  filename: string;
  mediaId: string;
  onPlaybackStart: (mediaId: string, stopFn: () => void) => void;
}
```

Component logic:
- `const audioRef = useRef<HTMLAudioElement>(null);`
- `const [isPlaying, setIsPlaying] = useState(false);`
- Play function: calls `onPlaybackStart(mediaId, stop)` first (this lets the parent pause any other playing audio), then `audioRef.current?.play()`, then `setIsPlaying(true)`.
- Pause function: `audioRef.current?.pause()`, `setIsPlaying(false)`.
- Stop function: `audioRef.current?.pause()`, `audioRef.current!.currentTime = 0`, `setIsPlaying(false)`.
- Add `onEnded` handler on the `<audio>` element: `() => setIsPlaying(false)`.
- Add a `useEffect` cleanup that pauses audio on unmount: `return () => { audioRef.current?.pause(); }`

Rendering:
```typescript
<div className="flex items-center gap-2 rounded-lg bg-card/60 border border-border/50 px-3 py-2">
  <audio ref={audioRef} src={src} onEnded={() => setIsPlaying(false)} preload="none" />
  {isPlaying ? (
    <button onClick={pause} className="h-8 w-8 flex items-center justify-center text-primary hover:bg-muted rounded-md transition-colors" aria-label="Pause audio">
      <Pause className="h-4 w-4" />
    </button>
  ) : (
    <button onClick={play} className="h-8 w-8 flex items-center justify-center text-muted-foreground hover:text-foreground hover:bg-muted rounded-md transition-colors" aria-label="Play audio">
      <Play className="h-4 w-4" />
    </button>
  )}
  <button onClick={stop} className="h-8 w-8 flex items-center justify-center text-muted-foreground hover:text-foreground hover:bg-muted rounded-md transition-colors" aria-label="Stop audio">
    <Square className="h-3.5 w-3.5" />
  </button>
  <span className="text-xs text-muted-foreground truncate flex-1">{filename}</span>
</div>
```

Export: `export function AudioPlayer`

Upload mutation:
```typescript
const uploadMutation = useMutation({
  mutationFn: (file: File) => {
    const formData = new FormData();
    formData.append('file', file);
    formData.append('element_id', elementId);
    return api.uploadMedia(projectId, formData);
  },
  onSuccess: () => {
    queryClient.invalidateQueries({ queryKey: QUERY_KEYS.ELEMENT_MEDIA(elementId) });
    setError(null);
  },
  onError: (err: Error) => {
    setError(err.message);
  },
});
```

Event handlers:
- `handleDragOver`: `e.preventDefault(); setIsDragOver(true);`
- `handleDragLeave`: `setIsDragOver(false);`
- `handleDrop`: `e.preventDefault(); setIsDragOver(false); const files = Array.from(e.dataTransfer.files); files.forEach(file => uploadMutation.mutate(file));`
- `handleFileSelect`: from `<input>` onChange: `const files = Array.from(e.target.files || []); files.forEach(file => uploadMutation.mutate(file)); e.target.value = '';` (reset input so same file can be re-selected)
- `handleClick`: `fileInputRef.current?.click();`

Rendering:
```typescript
<div
  onDragOver={handleDragOver}
  onDragLeave={handleDragLeave}
  onDrop={handleDrop}
  onClick={handleClick}
  className={`mt-3 border-2 border-dashed rounded-lg p-4 text-center transition-colors cursor-pointer min-h-[80px] flex flex-col items-center justify-center gap-1 ${
    isDragOver
      ? 'border-primary bg-primary/5'
      : 'border-border hover:border-border-strong'
  }`}
>
  <input
    ref={fileInputRef}
    type="file"
    accept="image/jpeg,image/png,image/webp,audio/mpeg,audio/wav,audio/x-m4a"
    multiple
    onChange={handleFileSelect}
    className="hidden"
  />
  {uploadMutation.isPending ? (
    <>
      <Loader2 className="h-6 w-6 text-primary animate-spin" />
      <span className="text-xs text-muted-foreground">Uploading...</span>
    </>
  ) : error ? (

  ) : isDragOver ? (
    <>
      <Upload className="h-6 w-6 text-primary" />
      <span className="text-xs text-primary font-medium">Drop to upload</span>

      <Upload className="h-6 w-6 text-muted-foreground/40" />

    </>
  )}
</div>
```

**2. Update `frontend/src/components/Breakdown/AssetsPanel.tsx`:**

Add `useRef`, `useCallback` to react imports (if not already present).

Add audio overlap prevention refs and callback:
```typescript
const stopCurrentAudioRef = useRef<(() => void) | null>(null);

const handlePlaybackStart = useCallback((mediaId: string, stopFn: () => void) => {
  // Stop any currently-playing audio before starting new one
  if (stopCurrentAudioRef.current) {
    stopCurrentAudioRef.current();
  }
  stopCurrentAudioRef.current = stopFn;
}, []);
```

Pass the callback to each `AssetElementCard`:
```typescript
<AssetElementCard
  key={element.id}
  element={element}
  projectId={projectId}
  onPlaybackStart={handlePlaybackStart}
/>
```

This ensures that when a user clicks Play on any AudioPlayer, the `onPlaybackStart` callback first calls the `stop()` function of the previously-playing audio (if any), then stores the new audio's stop function. This prevents multiple audio files from playing simultaneously (Pitfall 5 from RESEARCH.md).
