Create `frontend/src/components/Books/AgentPipelineTree.tsx` as a standalone component with these parts:

**Data fetching (3 queries):**
- `useQuery({ queryKey: [QUERY_KEYS.PIPELINE_MAP], queryFn: () => api.getPipelineMap() })` for pipeline map entries
- `useQuery({ queryKey: [QUERY_KEYS.AGENTS], queryFn: () => api.getAgents() })` for agent metadata (name, color, is_active)
- `useQuery({ queryKey: [QUERY_KEYS.TEMPLATE('short_movie')], queryFn: () => api.getTemplate('short_movie') })` for human-readable phase/subsection names

**Client-side data grouping:**
Build a `buildTreeData` function that:
1. Creates an `agentMap: Map<string, Agent>` from the agents array
2. Groups pipeline entries by `phase` then by `subsection_key`
3. Uses `templateConfig.phases` to get the ordering and human-readable names
4. Filters out entries where the agent no longer exists in agentMap
5. Returns an array of `TreePhase` objects (see Research pattern 1 for the exact types):

```typescript
interface TreePhase {
  phase: string;
  phaseName: string;
  subsections: TreeSubsection[];
}

interface TreeSubsection {
  key: string;
  name: string;
  agents: TreeAgent[];
}

interface TreeAgent {
  agentId: string;
  name: string;
  color: string;
  confidence: number;
  isActive: boolean;
}
```

**Collapsible state:**
- `const [expandedPhases, setExpandedPhases] = useState<Set<string>>(new Set())` -- all collapsed by default
- `togglePhase(phaseId)` toggles a phase in/out of the Set (see Research pattern for the existing codebase convention)

**Toggle mutation:**
```typescript
const toggleMutation = useMutation({
  mutationFn: ({ agentId, isActive }: { agentId: string; isActive: boolean }) =>
    api.updateAgent(agentId, { is_active: isActive }),
  onSuccess: () => {
    queryClient.invalidateQueries({ queryKey: [QUERY_KEYS.AGENTS] });
    // No PIPELINE_MAP invalidation needed -- is_active is not a semantic field,
    // agent stays in map but gets filtered at review time
  },
});
```

**Rendering structure:**
```
<div className="mt-6">
  <div className="flex items-center gap-2 mb-3">
    <Network className="h-4 w-4 text-muted-foreground" />
    <h3 className="text-sm font-semibold text-foreground">Pipeline Map</h3>
  </div>

  {/* Loading state */}
  {isLoading && <LoadingSkeleton />}

  {/* Empty state -- show when loaded but no entries */}
  {!isLoading && treeData.length === 0 && <EmptyState />}

  {/* Tree */}
  {treeData.map(phase => (
    <div key={phase.phase}>
      {/* Phase header -- clickable to expand/collapse */}
      <button onClick={() => togglePhase(phase.phase)} className="...flex items-center gap-1.5 w-full text-left py-1.5">
        {expandedPhases.has(phase.phase)
          ? <ChevronDown className="h-3.5 w-3.5 text-muted-foreground" />
          : <ChevronRight className="h-3.5 w-3.5 text-muted-foreground" />}
        <span className="text-sm font-medium text-foreground">{phase.phaseName}</span>
        <span className="text-xs text-muted-foreground ml-auto">
          {phase.subsections.reduce((sum, s) => sum + s.agents.length, 0)} agent(s)
        </span>
      </button>

      {/* Subsections -- only render when phase is expanded */}
      {expandedPhases.has(phase.phase) && (
        <div className="ml-4 mt-1 space-y-2 mb-2">
          {phase.subsections.map(sub => (
            <div key={sub.key}>
              <p className="text-xs font-medium text-muted-foreground mb-1">{sub.name}</p>
              <div className="flex flex-wrap gap-1.5">
                {sub.agents.map(agent => (
                  <AgentToggleBadge
                    key={agent.agentId}
                    agent={agent}
                    onToggle={(agentId, newState) =>
                      toggleMutation.mutate({ agentId, isActive: newState })}
                    isToggling={toggleMutation.isPending}
                  />
                ))}
              </div>
            </div>
          ))}
        </div>
      )}
    </div>
  ))}
</div>
```
