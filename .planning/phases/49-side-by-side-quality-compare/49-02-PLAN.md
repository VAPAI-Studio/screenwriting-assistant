## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-49-SC | Tampering | npm installs | mitigate | No new dependency (D-49 "no new dependency", reuses Radix Dialog + Button + lucide-react already present); no install task → gate N/A |
