<objective>
Replace the arrow-button shot reorder controls with drag-and-drop using @hello-pangea/dnd (React 18 compatible, maintained fork of react-beautiful-dnd).

Purpose: Arrow buttons are clunky for reordering; drag-and-drop is the expected UX for a production breakdown tool. The backend reorder endpoint already exists. This plan installs the library, wraps each SceneGroup in a Droppable, makes each ShotRow a Draggable, and calls the existing reorderMutation on drop.

@hello-pangea/dnd key API (React 18 compatible, drop-in for react-beautiful-dnd):
```typescript
import { DragDropContext, Droppable, Draggable, DropResult } from '@hello-pangea/dnd';

// DragDropContext wraps the whole draggable area, receives onDragEnd
// Droppable wraps a list, receives droppableId
// Draggable wraps each item, receives draggableId and index

// DropResult shape:
interface DropResult {
  draggableId: string;  // shot.id
  source: { droppableId: string; index: number };
  destination: { droppableId: string; index: number } | null;
}
```
