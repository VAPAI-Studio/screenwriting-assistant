    **1. Create `frontend/src/lib/textHighlight.ts`** -- a pure function module with zero React dependency:

    Export these types:
    ```typescript
    export interface ElementMatch {
      elementId: string;
      elementName: string;
      category: BreakdownCategory;
    }

    export interface TextSegment {
      type: 'text' | 'highlight';
      content: string;
      match?: ElementMatch;
    }
    ```

    Export `escapeRegex(str: string): string` -- escapes regex metacharacters: `str.replace(/[.*+?^${}()|[\]\\]/g, '\\$&')`

    Export `buildHighlightSegments(text: string, elements: BreakdownElement[]): TextSegment[]`:
    - If `!elements.length || !text`, return `[{ type: 'text', content: text }]`
    - Filter to non-deleted elements with non-empty trimmed names: `elements.filter(e => e.name.trim().length > 0 && !e.is_deleted)`
    - Sort by name length descending (longest match first to avoid "John" matching inside "John Smith")
    - Build a case-insensitive lookup Map keyed by `name.toLowerCase()` to the element
    - Build regex pattern: `sorted.map(e => escapeRegex(e.name)).join('|')` wrapped in word boundaries: `new RegExp('\\b(' + pattern + ')\\b', 'gi')`
    - Iterate with `regex.exec(text)` loop, building segments array alternating between plain text (sliced from lastIndex to match.index) and highlight segments (with the ElementMatch from the lookup map via `matchedText.toLowerCase()`)
    - After loop, append remaining text if `lastIndex < text.length`
    - Return segments array

    Import `BreakdownCategory` and `BreakdownElement` from `../../types` (adjust path: this file is in `lib/`, so import from `../types`).
